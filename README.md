
# 시골쥐의 어디가쥐 

[프로젝트 사이트](https://www.countryrat.site)

## :computer:소개

 "시골쥐의 어디가쥐"는 사용자가 자신의 여행 기록을 작성하고, 다른 사람들의 여행 이야기도 공유하며 함께 즐길 수 있는 웹사이트입니다. 이 웹은 여행지의 추억을 기록하고, 다른 사람들과 그 경험을 나누는 플랫폼을 제공합니다.
## :rocket:주요 기능

* 여행 기록 작성
* 게시글 열람
* 사진 및 경로 공유
* 좋아요 기능
